# Gov Doc Parser

A Python library for extracting structured information from Indian government documents using AI.

## Overview

Gov Doc Parser provides an easy-to-use interface for extracting information from various Indian government documents like Aadhaar cards, PAN cards, and Form 16. It uses advanced AI models to accurately parse and structure the information from document images.

## Features

- Support for multiple document types:
  - Aadhaar Card (Front and Back)
  - PAN Card
  - Form 16
- Flexible AI model configuration
- Async and sync APIs
- JSON output format
- Environment variable support
